## 滑动窗口算法

1，设置左右边界,存储字符和索引的容器map，取字符串长度

```java
int left , right = 0;
Map<Charactor,Integer> window = new HashMap<>();
int len  = s.length();
int max_len = 0;
```

2.遍历字符串，第一次遍历的元素就添加到map中

```java
while(right < len){
    char currentchar = s.charAt(right);
    map.put(currentchar,right);//添加索引
    max_len = Math.max(max_len,right-left+1);
    right++;
}
```
