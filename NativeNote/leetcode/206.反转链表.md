### 反转链表如何做：

双指针法：

一个前驱节点 prev ， 一个现节点 curr  ， 一个后驱节点 next

要把链表的箭头 反过来 ， 并把后去节点放到后面防止断链



核心思路：

把链表看成   1 -> 2 -> 3-> null    反转后就是    null <- 1 <- 2 <- 3 

next  = curr.next

curr.next = prev; //箭头反转

//移动指针

prev = curr;

curr = next;



//初始条件 

curr= head；

prev = null；
