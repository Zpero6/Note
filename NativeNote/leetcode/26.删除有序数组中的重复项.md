### 删除重复数

由于数组是排好序的，越往后 数字是越大的，只要找见不重复的就把它添加到 前一个地方的前面

#### 双指针法

从数组索引1 开始 ， 然后设置慢指针 ， 再设计一个快指针 ， 都从1开始；

快指针用来寻找数字 ， 满指针用来记录取代位置



```java
class Solution {
    public int removeDuplicates(int[] nums) {
        // set会增加空间复杂度， 而且contains（）方法开销比较大
        // 使用快慢指针
        if (nums.length == 0) {
            return 0;
        }
        int slow = 1;
        for (int fast = 1; fast < nums.length; fast++) {
            if (nums[fast] != nums[slow - 1]) {
                nums[slow] = nums[fast];
                slow++;
            }
        }
        return slow;
    }
}
```
