### 移除元素

快慢指针

快指针负责遍历寻找和val值相同的元素

慢指针负责定位修改位置

因为要找和val值相同的元素 ， 要从0开始遍历 ， 当不是val就往前走 ， 当是val时slow就停下，定位到剔除位置。

```java

import java.util.Arrays;

class Solution {
    public int removeElement(int[] nums, int val) {
       
    	int slow = 0;
        for(fast = 0; fast<nums.length;fast++){
            if(nums[fast] !=  val){
                nums[slow] = nums[fast];
                slow++;
            }
        }
        return slow;
   }
}

```
