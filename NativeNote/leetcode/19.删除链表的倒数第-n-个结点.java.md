#### 思路： 快慢指针

倒数第n位 可以理解为两根指针差了 n-1 位 ， 要让快指针比慢指针多n位 ， 这时候慢指针把它的下一位越过链接，然后两者之间就会差n-1位 ， 达到删除第n个元素

```java
class Solution {
    public ListNode removeNthFromEnd(ListNode head, int n) {

        // 1. 创建虚拟头结点 (Dummy Node)
        ListNode dummy = new ListNode(0);
        dummy.next = head;

        ListNode fast = dummy;
        ListNode slow = dummy;

        // 2. 快指针先移动 N 步，拉开差距
        // N 步后，fast 和 slow 之间有 N 个结点的间隔
        for (int i = 0; i < n; i++) {
            fast = fast.next;
        }

        // 3. 同时移动双指针，直到快指针到达链表末尾
        // 当 fast.next == null 时，fast 位于链表的最后一个结点
        // 当 fast == null 时，fast 位于链表的末尾（即 null）
        while (fast.next != null) {
            fast = fast.next;
            slow = slow.next;
        }

        // 4. 执行删除操作
        // 此时 slow 停在了倒数第 N 个结点的前一个位置
        // slow.next 就是要删除的结点
        slow.next = slow.next.next;

        // 返回虚拟头结点的下一个结点，即新的链表头
        return dummy.next;
    }
}
```
